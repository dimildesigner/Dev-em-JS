Sinais de comparação lógicos

Sinais de comparação lógicos sempre terão saídas lógicas, ou seja, ou será verdadeiro ou falso.

1. Igualdade abstrata (==)

5 == '5' - true

2. Igualdade estrita (===)

5 === '5' - false

3. Diferença abstrata (!=)

5 != '5' - true


Relacionais

maior (>)

5 > 6 - false

menor (<)

5 < 5 - false

maior ou igual (>=)

5 < 5 - true

maior ou igual (<=)

5 <= 5 - false

SINAIS DE COMPRAÇÃO LÓGICOS

SINAIS DE COMPARAÇÃO LÓGICOS SEMPRE VÃO TER SAIDAS LÓGICAS - true / false

1 - igualdade comparação / (==) abstrata

5 == '5' - true

2 - Igualdade comparação / (===) estrita

5 === '5' - false

3 - Diferença abstrata (!=)

5 != '5' - false

4 - Diferença estrita (!==)

5 !== '5' - true

Relacionais

maior (>)

5 > 6 - false

menor (<)

5 < 5 - false

maior ou igual (>=)

5 >= 5 - true

menor ou igual (<=)

5 <= 5 - true

SINAIS DE COMPARAÇÃO ARITMÉTICO

SINAIS DE COMPARAÇÃO ARIMÉTICOS - TEREMOS SAIDAS NÚMERICAS

SOMA (+)

5 + 5 = 10

SUBTRAÇÃO (-)

5 - 5 = 0

MULTIPLICAÇÃO (*)

5 * 5 = 25

DIVISÃO (/)

5 / 5 = 1

MÓDULO % RESTA DA CONTA DE DIVISÃO

5 % 5 = 0
